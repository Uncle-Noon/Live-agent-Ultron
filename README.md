1. Add your gemini api key  and groq api key in <code>.env</code> file in this format: <br>
GEMINI_API_KEY= YOUR_GEMINI_API_KEY <br>
GROQ_API_KEY= YOUR_GROQ_API_KEY <br> 
2. Open terminal in root directory and do <code>npm i</code> to download packages and dependencies <br>
3. Do <code>npm run dev</code> in terminal and make sure it's running on port 3000 <br>
4. Double click the <code>chat-ui.html</code> file to open it in browser <br>
